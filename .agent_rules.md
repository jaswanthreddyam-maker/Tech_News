# Workspace Rules

Canonical repository:
`D:\tech_news`

Before making ANY modification:

1. Verify Docker bind mounts.
2. Verify current working directory.
3. Verify active Git repository.
4. Verify IDE path matches Docker bind mount.

If any mismatch exists:
STOP.
Report the mismatch.
Do not edit any file.

Never assume the workspace.
Always verify it first.
